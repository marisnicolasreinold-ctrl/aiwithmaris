# Themenplan — Täglicher Blog

So funktioniert es:
- **Montag, Mittwoch, Donnerstag, Samstag, Sonntag:** Evergreen-Artikel —
  das Skript nimmt das oberste offene Thema aus der Liste unten.
- **Dienstag & Freitag:** News-Kommentar — das Skript sucht eine aktuelle
  KI-Nachricht und schreibt Maris' Einordnung für den Mittelstand dazu.
- Läuft die Liste leer, generiert das Skript automatisch zehn neue Themen
  nach und trägt sie hier ein.
- Veröffentlichte Themen wandern mit Datum in den Abschnitt „Veröffentlicht".

Du kannst hier jederzeit Themen ergänzen, umsortieren oder löschen — das
oberste offene Thema ist immer das nächste.
